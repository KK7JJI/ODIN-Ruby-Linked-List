# ODIN-Ruby-Linked-List
The project details are located:
[Project: Linked Lists](https://www.theodinproject.com/lessons/ruby-linked-lists)

In breif the exercise involves implementing a linked list datastructure in Ruby having the following:

### Classes:
- LinkedList
- Node

### Methods
- #append
- #prepend
- #size
- #head
- #tail
- #at(index)
- #pop
- #contains?(value)
- #index(value)
- #to_s
- #insert_at(index, *values)
- #remove_at(index)
